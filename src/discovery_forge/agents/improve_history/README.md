# Researcher Prompt Improvement History

This directory stores dated prompt-improvement records for `researcher.md`.

Use one directory per improvement day:

```text
src/discovery_forge/agents/improve_history/<day>/plan.md
src/discovery_forge/agents/improve_history/<day>/applied.md
```

If the prompt is improved multiple times on the same day, overwrite that day's `plan.md` and `applied.md` with the latest state.
